# 🚀 Космическая стрелялка

Небольшая аркадная игра на Python (pygame), где нужно уничтожать врагов, собирать бонусы и выживать как можно дольше!

Сделано **Vlad Games**.

## 🎮 Как играть

| Клавиша | Действие |
|---|---|
| `←` `→` или `A` `D` | Двигать корабль |
| `ПРОБЕЛ` | Стрелять |
| `ENTER` | Подтвердить ник при первом запуске |

- Уничтожай врагов и набирай очки
- Не давай врагам долетать до низа экрана — за это теряешь жизни (всего их 3)
- Продержись **20 секунд**, и начнётся **2-й уровень** — враги станут сильнее, крупнее и быстрее (у них появится полоска здоровья)
- Враги иногда роняют бонусы:
  - 🛡️ **Щит** (синий) — временная защита от урона
  - 🔺 **Тройной выстрел** (оранжевый) — три пули веером вместо одной

## ✨ Особенности

- Собственное меню с кнопками "Играть" и "Гайд"
- Ползунки громкости музыки и звуков
- Ник игрока запоминается между запусками
- Музыка и звуковые эффекты сгенерированы прямо в коде, без внешних аудиофайлов
- Своя иконка
- В версии 2.0.0 добавлена система модов! 
## 🛠️ Как запустить

Нужен Python 3.10+ и pygame:

\`\`\`bash
pip install pygame
python space_shooter.py
\`\`\`

## 📦 Скачать готовую игру

Собранный `.exe` для Windows — в разделе [Releases](../../releases) этого репозитория, запускать без установки Python.

## 🔨 Как собрать .exe самому

\`\`\`bash
pip install pyinstaller
pyinstaller --onefile --windowed --icon="game_icon.ico" --version-file="version_info.txt" --name "SpaceShooter" space_shooter.py
\`\`\`

Готовый файл появится в папке `dist`.

## 📄 Файлы в репозитории

- `space_shooter.py` — исходный код игры
- `game_icon.ico` — иконка для сборки
- `version_info.txt` — информация об издателе для PyInstaller
